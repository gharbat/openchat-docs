### Make sure docker is installed

You can download docker [here](https://www.docker.com/products/docker-desktop) if you don't have it installed already.
currently OpenChat is only supported on Mac and Linux. Windows support is coming soon.

### Installing

<AccordionGroup>
    <Accordion icon="github" title="Clone OpenChat locally">
        Clone the OpenChat repository locally by running the following command in your terminal:

        ```bash
        git clone https://github.com/openchatai/OpenChat
        ```

    </Accordion>
    <Accordion icon="rectangle-terminal" title="Update the environment variables">
        Update common.env with your keys:

        ```
        OPENAI_API_KEY=# you can get it from your account in openai.com
        PINECONE_API_KEY=# you can get from "API Keys" tab in pinecone
        PINECONE_ENVIRONMENT=# you can get it after creating your index in pinecone
        PINECONE_INDEX_NAME=# you can get it after creating your index in pinecone
        ```
    </Accordion>

    <Accordion icon="rectangle-terminal" title="Run the installation command">
        Run the following command in your terminal to install OpenChat:

        ```bash
        make install
        ```

        This will install all the dependencies and start the server. You can access the server at http://localhost:8000
    </Accordion>
</AccordionGroup>
